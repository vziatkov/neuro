Task
Simulate a large 2D world with a high number of objects.
Render only visible objects within the viewport.
Ensure smooth camera navigation and responsiveness.
Avoid creating all objects in Pixi scene at once.
Demonstrate performance thinking, not visual polish.
Solution
World is treated as data, not display objects.
Only visible elements are instantiated and rendered.
Lightweight simulation demonstrates update + interaction layers.
Visual effects represent dynamic object lifecycle.
Architecture is designed for scaling to large datasets.
