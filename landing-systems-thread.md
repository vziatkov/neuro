# Systems Thread — why these experiments feel related

🚀 [https://vziatkov.github.io/neuro/landing-with-art-cards.html](https://vziatkov.github.io/neuro/landing-with-art-cards.html)

If you look at the viewer, it solves a very practical problem:
**rendering large worlds efficiently**.

But the same idea shows up across my other experiments too.

On the landing page, each card explores a slightly different angle:

- particle systems that react instead of animate
- grid-based structures that reveal patterns over time
- trace systems where motion leaves memory
- small “formations” emerging from simple rules

The common thread is simple:

> visuals are not predefined — they are the result of a system

In the world viewer, this becomes:

- data → projection → visibility

In the particle experiments, it becomes:

- rules → interaction → emergence

Different surfaces. Same thinking.

I’m interested in systems where:

- behavior drives visuals
- scale doesn’t break the experience
- complexity emerges from simple rules

Some of these are small experiments.
Some can evolve into real products.

Curious what resonates more with you:
the **engineering side** or the **emergent behavior side** 👇
