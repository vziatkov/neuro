# Forest Supply Chain — Simulation Example (Supply Chain Digital Twin)

Пример скилла: **архитектура цифрового двойника бизнеса** — экономическая сеть узлов и потоков с рисками, энергией, географией, двумя режимами симуляции и декомпозицией прибыли. Модель пригодна для Monte Carlo, поиска +10% эффективности, живой интерактивной карты и стратегических сценариев (what-if).

## Содержимое

- **CONCEPT_MAP.md** — онтология и data model цифрового двойника:
  - Узлы, потоки, контракты, инвентарь, рынок и цены, транспорт, качество древесины, этапы производства и очереди между стадиями.
  - **Риски и неопределённость** (пожары, штормы, задержки, скачки цен) для Monte Carlo.
  - **Энергия и топливо** (сушка), **география** (координаты → карта потоков), **поведение менеджеров**, **калибровка на данных**.
  - **Два режима симуляции:** быстрый (интерактивная карта) и полный Monte Carlo; **автопоиск bottleneck**; **декомпозиция прибыли** (сырьё, логистика, производство, хранение); **стратегические сценарии** (новая сушильная линия, лесопилка).

## Контекст

Система обучения для оптимизации затрат лесного бизнеса. Модель представлена как граф: ноды (источники сырья, производство, переработка, продажи, логистика) и потоки между ними (сырьё, материалы, субпродукты, товары). На графе считаются метрики (throughput, загрузка, себестоимость, маржа) и запускаются симуляции при разных сценариях спроса, цен и решений (операционные / тактические / стратегические).

## Дальнейшие шаги

- Реализация графа в JSON (пример инстанса).
- **Simulation engine:** состояние + решения + контракты + риски + очереди; режимы **interactive** и **Monte Carlo**; расчёт декомпозиции прибыли и **автопоиск bottleneck**.
- Прототип **живой карты**: ползунки, граф на географической карте, сравнение сценариев, стратегические what-if.
- Калибровка параметров на реальных данных компании для предсказательной симуляции.
